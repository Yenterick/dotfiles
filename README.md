## Setup

Clone the repository:

```bash
git clone https://github.com/Yenterick/dotfiles.git ~/dotfiles
```

Create symlinks:

```bash
ln -s ~/dotfiles/.zshrc ~/.zshrc
ln -s ~/dotfiles/nvim ~/.config/nvim
```
